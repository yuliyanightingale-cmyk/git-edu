# Git Lesson

## Git Commannds

__git init__ - инициализация репозитория.

__git status__ - отображает текущее состояние репозитория: на какой ветке какие файлы изменились.

__git add__ - добавить файлы в репозиторий.

__git commit__ - зафиксировать текущее состояние.

__git log__ - вывести историю фиксаций (коммитов).

__git checkout__ - изменить текущую версию (перемещение между коммитами).


## Text formatting

*cours* - наклонный текст для выделения.

__bold__ - выделение текста жирным шрифтом для акцентирования внимания.

## Lists

Маркированный список:

* elem1
- elem2

Нумерованный список: 

1. elem1
2. elem2

## Tables

## Git commands GitHub

__Основные команды для GitHub:__

__git clone__ - склонировать удаленный репозиторий

__git push origin main__ - отправить изменения на GitHub

__git remote add__ - добавить удаленный репозиторий

__git pull request__ - запрос на слияние изменений
